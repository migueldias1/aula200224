Curso de CSS - Do Básico ao Avançado

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para controlar a apresentação e o layout de documentos HTML. Ele define como os elementos HTML devem ser exibidos na página, incluindo cores, fontes, espaçamento, posicionamento e muito mais.

Seletores

Seletores CSS são padrões usados para selecionar os elementos HTML aos quais você deseja aplicar estilos. Existem diversos tipos de seletores, incluindo seletores de tipo, de classe, de ID, combinadores, pseudo-classes e pseudo-elementos.

  • Seletor de tipo: >p { color: blue; }
  • Seletor de classe: .destaque { font-weight: bold; }
  • Seletor de ID: #paragrafo { font-style: italic; }
  • Combinador de descendência: div p { text-decoration: underline; }
  • Pseudo-classe: a:hover { color: red; }
  • Pseudo-elemento: p::first-letter { font-size: 150%; }

Propriedades e Valores

As propriedades CSS especificam as características que você deseja alterar, como cor, tamanho, margem, etc. Cada propriedade possui um valor associado que define como essa característica deve ser exibida.

  • Cor do texto: h1 { color: #FF0000; }
  • Tamanho da fonte: h2 { font-size: 24px; }
  • Margem superior: p { margin-top: 20px; }
  • Alinhamento de texto: div { text-align: center; }

Box Model

O modelo de caixa (box model) é um conceito fundamental do CSS que descreve como os elementos HTML são renderizados no navegador. Ele consiste em quatro áreas: conteúdo, preenchimento, borda e margem. Entender o box model é essencial para criar layouts precisos e responsivos.

    div {
width: 200px;
height: 100px;
padding: 20px;
border: 2px solid #000;
margin: 10px;
}

Layouts

CSS oferece várias técnicas para criar layouts complexos e responsivos. Isso inclui o uso de propriedades de posicionamento, como position, float, display, flexbox e grid, que permitem posicionar e organizar elementos na página de maneira eficiente.

    .container {
display: flex;
justify-content: space-around;
}

.item {
width: 100px;
height: 100px;
background-color: #333;
color: #fff;
text-align: center;
line-height: 100px;
}

Estilização Avançada

Além das propriedades básicas de estilo, CSS oferece recursos avançados para estilização, como gradientes, sombras, transições, animações e transformações. Esses recursos permitem adicionar efeitos visuais sofisticados às páginas da web.

    div {
background: linear-gradient(to right, #ff0000, #0000ff);
}

div {
box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);
}

button {
transition: background-color 0.3s ease;
}
button:hover {
background-color: #ff0000;
}

@keyframes example {
from {transform: scale(1);}
to {transform: scale(1.2);}
}
div {
animation: example 2s infinite alternate;
}

Media Queries

Media queries são usadas para aplicar estilos com base nas características do dispositivo, como largura da tela, altura da tela, orientação, resolução, etc. Isso permite criar layouts responsivos que se adaptam a diferentes dispositivos e tamanhos de tela.

    @media screen and (max-width: 600px) {
div {
    font-size: 14px;
}
}

Pré-processadores CSS

Pré-processadores CSS, como Sass, Less e Stylus, são ferramentas que estendem a sintaxe padrão do CSS, fornecendo recursos avançados, como variáveis, mixins, aninhamento, herança, entre outros. Essas ferramentas ajudam a tornar o código CSS mais modular, reutilizável e fácil de manter.

    $cor-primaria: #ff0000;

body {
background-color: $cor-primaria;
}

Frameworks CSS

Frameworks CSS, como Bootstrap, Foundation e Bulma, são conjuntos de estilos pré-definidos e componentes que podem ser utilizados para acelerar o desenvolvimento web. Eles fornecem grids, tipografia, formulários, botões e outros componentes comuns, facilitando a criação de interfaces consistentes e atraentes.

Exemplo de uso do Bootstrap:

    <!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
<title>Exemplo Bootstrap</title>
</head>
<body>

<div class="container">
<h1>Meu Site</h1>
<p>Este é um exemplo de layout usando Bootstrap.</p>
<button class="btn btn-primary">Botão</button>
</div>

</body>
</html>

Este curso abrange desde os conceitos básicos até as técnicas mais avançadas em CSS, preparando os alunos para criar páginas web profissionais e visualmente impressionantes.
