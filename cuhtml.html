Curso de HTML - Do Básico ao Avançado

O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar documentos na web. Ele define a estrutura e o conteúdo de uma página web através de elementos e tags.

Estrutura Básica de um Documento HTML

Todo documento HTML começa com <!DOCTYPE html>, seguido pela tag <html>, que envolve todo o conteúdo da página. O elemento <head> contém metadados, como título da página (<title>), conjunto de caracteres (<meta charset="UTF-8">), links para estilos CSS, scripts JavaScript e outras informações importantes. O elemento <body> contém o conteúdo visível da página, como texto, imagens, links, tabelas, formulários, etc.

Tags e Elementos HTML

Tags HTML são usadas para definir e estruturar o conteúdo da página. Uma tag HTML é composta por um nome e pode conter atributos para fornecer informações adicionais.

  • <p> para parágrafos
  • <h1> a <h6> para cabeçalhos
  • <a> para links
  • <img> para imagens
  • <ul> e <ol> para listas
  • <div> e <span> para agrupamento e estilização de elementos, entre outros

Formatação de Texto

Tags como <strong>, <em>, <u>, <s>, <b>, <i>, <sup>, <sub> são usadas para formatar o texto, fornecendo ênfase, itálico, sublinhado, tachado, negrito, itálico, sobrescrito, subscrito, etc. Também há tags para representar linhas quebradas (<br>) e espaços (&nbsp;).

Listas

HTML suporta dois tipos principais de listas:

  • Listas não ordenadas (<ul>): itens de lista com marcadores
  • Listas ordenadas (<ol>): itens de lista numerados

Os itens de lista são definidos pela tag <li> dentro das tags <ul> ou <ol>.

Links e Âncoras

A tag <a> é usada para criar links para outras páginas web, documentos, arquivos, etc. O atributo href na tag <a> especifica o URL de destino do link. Âncoras (<a> com href apontando para uma posição específica na mesma página) são usadas para navegar para seções diferentes de uma página.

Imagens

A tag <img> é usada para incorporar imagens em uma página web. O atributo src na tag <img> especifica o caminho para o arquivo de imagem. Outros atributos, como alt, são usados para fornecer texto alternativo para a imagem.

Tabelas

A tag <table> é usada para criar tabelas em HTML. As células de uma tabela são definidas pelas tags <tr> para linhas e <td> para células de dados. Tags adicionais, como <th> para cabeçalhos de tabela e <caption> para legenda, podem ser usadas para melhorar a semântica da tabela.

Formulários Avançados

Além dos elementos de formulário básicos, como <input> e <textarea>, HTML5 introduziu novos tipos de entrada, como <input type="date">, <input type="email">, <input type="range">, etc. Também há suporte para validação de formulários usando os atributos required, pattern, min, max, entre outros.

Multimídia

Além de imagens, HTML5 permite a incorporação de vídeo e áudio em uma página web usando as tags <video> e <audio>. Isso permite uma experiência multimídia mais rica para os usuários.

APIs de HTML5

HTML5 introduziu várias APIs poderosas que permitem interações avançadas com o navegador e dispositivos. Algumas das APIs populares incluem a API de Geolocalização, a API de Armazenamento Web (LocalStorage e SessionStorage), a API de Histórico de Nave
